# Eureka SAS Bot

## Overview
The Eureka SAS Bot is a project designed to enhance workflows and automate tasks using the SAS programming language. It provides various functionalities to streamline processes and improve productivity.

## Installation
To install the Eureka SAS Bot, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/eurekasas10/eureka-sas-bot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd eureka-sas-bot
   ```
3. Install the necessary dependencies from `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To use the Eureka SAS Bot, run the following command:
```bash
python main.py
```

Feel free to customize the configuration files as needed to tailor the bot's functionality to your specific workflows.
